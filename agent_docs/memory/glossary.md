# Glossary

Project-specific terms, acronyms, and names.

- **Bedrock** — the CLI and project-cockpit workflow in this repository.
- **Memory** — curated, durable project knowledge.
- **Work** — current focus, open loops, and recommended next actions.
- **Views** — generated human inspection artifacts such as the static site and compact index.
- **History** — lightweight chronology of releases, backfills, and major milestones.
- **Evidence** — imported or extracted material that is useful context but not canonical truth.
- **Legacy vault** — the older `./agent-knowledge/` tree that predates the active `./bedrock/` layout.
